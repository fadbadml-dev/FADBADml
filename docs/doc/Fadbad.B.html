type t = {
   mutable operator : op;
   mutable operands : t array;
   mutable rc : int;
   mutable value : Op.t;
   derivatives : D.t;
}

Type of elements

  • operator : the operation described by this node
  • operands : the arguments of the operator
  • rc : the reference counter; it is always equal to the number of nodes that points to this sub-expression plus one
  • value : the current value of the sub-expression
  • derivatives : the derivative of the root expression with respect to this sub-expression
